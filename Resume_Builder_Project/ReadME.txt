In this project we are building our very own Resume Builder Website. We can build any resume using this website.

Here are using HTML, CSS, Bootstrap, JS, Google Firebase - Realtime Database, PostgreSQL.

1. We used HTML, CSS, JS for frontend designing.

2. webcam.js for clicking of image in realtime using webcam.

3. Realtime Database for Storage of Image in Image URI format.

4. PostgreSQL for storing non Image data.
